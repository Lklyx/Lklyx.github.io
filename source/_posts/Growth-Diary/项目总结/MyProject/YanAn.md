# 还没开始就遇到坑！

1. 开始的时候用什么css预处理器，在网上找了半天，我后面决定用sass，原因很简单，它牛*！

   直接去他的官网(Sass)[https://www.sass.hk/install/]下载！下载了以后认真看安装步骤，一步一步走就完事了。

2. 在项目中安装sass

   ```shell
   npm install --save-dev sass-loader
   //sass-loader依赖于node-sass
   npm install --save-dev node-sass
   ```
